What's the future for this extension?

As described in the explainer above, with the introduction of EIP-1559 the fees will be split between two variables: the base fee and priority fee (also known as miner tip). At present, immediately after the upgrade, how some of these variables will play out, especially the priority fee is still uncertain.

For now, the extension will show the current base fee (or a recommended base fee). So, you can get a better understanding of the overall congestion of the network. The recommended base fee is currently set to be 2 times the existing base fee. This provides enough certainty that your transaction will be included according to the base fee. If you over pay the base fee, you will be refunded. So don't worry about overpaying. However, it's important to note, you need more than just the base fee to get your transaction included, you also need to choose the priority fee.

With regards to the priority fee, this is still uncertain and estimating what a good priority fee is subject to a lot of speculation. With this uncertainty, I'd rather wait and see how the network stabilizes on these amounts. I will be keeping a close on 3rd party providers and/or consider writing a simpler oracle to help users determine how much to price their priority fee.

I will be rapidly updating this extension as new information appears. I appreciate the patience & hope to make sure the transition is smooth for everyone. For the latest updates follow me on Twitter: @nieldlr. We might get information that helps users before I get time to update the extension. So please do consider keeping an eye there.
